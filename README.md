# KINEGRAM eMRTD Connector iOS SDK

iOS SDK for KINEGRAM eMRTD verification using the v2 WebSocket protocol.
The *eMRTD Connector* enables the [Document Validation Server (DocVal)][docval]
to read and verify eMRTD documents through a secure WebSocket connection.
